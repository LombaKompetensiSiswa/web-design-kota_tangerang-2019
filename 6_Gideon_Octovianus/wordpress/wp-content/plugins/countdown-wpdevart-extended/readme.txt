Countdown Extended is an fresh and extended version of countdown timer. You can easily create countdown and countup timers to your website. 

== Description ==

https://www.youtube.com/watch?v=1zaWuLZaWCc

Now you can easily create any Countdown or CountUp you need. There are a lot of new features that you will like. We added popup and some other important things for our extended version. 
There are also nice and handy themes that you can use.

== Installation ==

### First you need to download the plugin ZIP file.  

1. Log in to your admin panel.   
2. Go to Plugins page, click on Add New, then click "Upload Plugin" .    
3. Then click "Choose file" then select the plugin .zip file. 
4. Then install and activate the plugin.
